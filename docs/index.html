‚úÖ Canonical URLs Active

All QuestFoundry schemas are available at canonical URLs:

https://questfoundry.liesdonk.nl/schemas/{schema-name}.schema.json

Examples

  • https://questfoundry.liesdonk.nl/schemas/hook_card.schema.json
  • https://questfoundry.liesdonk.nl/schemas/tu_brief.schema.json
  • https://questfoundry.liesdonk.nl/schemas/canon_pack.schema.json

Alternative Access

  • GitHub Raw: https://raw.githubusercontent.com/pvliesdonk/questfoundry-spec/main/03-schemas/
  • GitHub Releases: Download schema bundles from releases
  • Python Package: pip install questfoundry-py

Validation with ajv-cli

$ npm install -g ajv-cli $ ajv validate \ -s https://questfoundry.liesdonk.nl/schemas/hook_card.schema.json \ -d your-hook.json

Python Validation

$ pip install jsonschema requests $ python -c " import requests, jsonschema, json schema = requests.get('https://questfoundry.liesdonk.nl/schemas/hook_card.schema.json').json() instance = json.load(open('your-hook.json')) jsonschema.validate(instance, schema) print('‚úì Valid!') "

Using questfoundry-py

$ pip install questfoundry-py from questfoundry.resources import get_schema schema = get_schema('hook_card')
