# Source Serif 4

**Status:** Fonts not yet installed (download required)

## Download Instructions

1. Visit: https://github.com/adobe-fonts/source-serif/releases
2. Download the latest release (OTF format)
3. Extract the following files to this directory:
   - `SourceSerif4-Regular.otf`
   - `SourceSerif4-Italic.otf`
   - `SourceSerif4-Bold.otf`
   - `SourceSerif4-BoldItalic.otf`
4. Copy `LICENSE.txt` from the source repository to this directory
